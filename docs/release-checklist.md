# Release checklist

- [ ] CI matrix (SQLite compatibility) passing
- [ ] Unit tests and E2E tests passing
- [ ] Documentation updated (README / docs/implementation-artifacts)
- [ ] CHANGELOG entry added
- [ ] Smoke workflow configured and green
- [ ] Telemetry/metrics enabled or documented
- [ ] Version compatibility notes for SQLite included

Use this checklist in PRs that modify persistence, ingestion, or core algorithms.
