## Notable Discoveries

### Issue #1: JSON Parsing from Mixed Output
**Problem**: CLI outputs status messages mixed with JSON, causing `json.loads(result.stdout)` to fail.

**Solution**: Created `_extract_json_from_cli_output()` helper that uses regex to extract JSON block:
```python
json_match = re.search(r"\{.*\}", output, re.DOTALL)
```

### Issue #2: Tamper Operation Breaking Structure
**Problem**: Original tamper operation (`snapshot_path.write_text("corrupted,data\n1,2\n")`) caused column validation failure (exit 2) before checksum check (exit 1).

**Solution**: Modified tamper to preserve CSV structure:
```python
tampered_df = df.copy()
tampered_df.loc[0, "close"] = 999.99  # Change data, not structure
tampered_df.to_csv(snapshot_path, index=False)
```

This ensures checksum mismatches while maintaining valid columns, achieving exit 1 (WARN) as expected.
